To run the website, Install XAMPP and extract the “ecommerce-prod.rar” file in to the htdocs folder of the xampp installation.
Open a browser and navigate to localhost/phpMyAdmin
Create a database name “ecommerce”
Import the db file inside the folder named “ecommerce.sql”
Open the db.php file and edit the connection string if necessary.
Now the website can be accessed from a web browser through the url, localhost/ecommerce-prod
